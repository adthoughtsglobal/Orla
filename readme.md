# Orla
Orla is the TUI like chat application built with OriginChats. It's an OriginChats client unlike any other.

## Available features
Orla is not a fully fledged OriginChats client that would support all the features as they are released, it is rather a more stable, optimal and reliable alternative that supports basic functionality and additional convinient features.

## Using Orla
### Panes
Orla comes with three *panes*. Panes in Orla are display sections that's used for different purposes.
1. Interactive logs pane: AKA messages pane displays messages, actions and other important interactive logs.
2. Orla commands and logs pane: where you navigate orla and see updates and context.
3. Information search Pane: a place where you can display different kinds of information and search them.

### Themes
Orla comes with a very powerful theme engine. You can mix multiple theme stylesheets to create amazing custom themes that change your experience in Orla.
**Tip**: Try `theme cute-dark`! or perhaps `theme centered-server-bar`. Remove everything with `theme remove all`
